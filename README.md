# E-commerce Platform (Full Stack + Android)

Full Stack e-commerce platform with Stripe payment integration and Android app connectivity.

## Features
- Product listing and management  
- Shopping cart functionality  
- Stripe payment processing (test environment)  
- REST-like API for client-server communication  
- Android app integration  

## Tech Stack
- Backend: PHP  
- Frontend: JavaScript, HTML, CSS, Bootstrap  
- Mobile: Java (Android)  
- Database: MySQL  
- Integrations: Stripe, Firebase  

## Project Structure
- `/assets` → Frontend resources (CSS, JS, images)  
- `/php` → Backend API endpoints  
- `/firebase` → Firebase integration  
- `/stripe-php-master` → Stripe SDK  
- `/phpmailer-master` → Email service library  

## Setup
1. Clone the repository  
2. Configure environment variables (`.env`)  
3. Run on local server (XAMPP, etc.)  

## Payments
This project uses Stripe in test mode for demonstration purposes.


## 📌 Notes
UI is based on a responsive template, extended with custom functionality and backend integration.
